HTTP`S` secure를 의미한다.

HTTPS는 HTTP에 SSL(Secure Socket Layer)이 추가되어있다.

주요한 차이점은 보안이다.

SSL 또는 TLS 프로토콜을 사용하여 데이터를 암호화한다. 데이터를 전송하는 동안 제 3자가 감시하거나 조작하는 것을 방지하고 사용자의 개인정보를 보호할 수 있게 해준다.

HTTP는 80번 포트를 사용하며 HTTPS 443번 포트를 사용한다.
